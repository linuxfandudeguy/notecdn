NoteCDN Script Fetching Documentation

Welcome to the NoteCDN script fetching documentation. NoteCDN fetches scripts using a path-based method similar to jsDelivr.

How to Use NoteCDN

To fetch scripts using NoteCDN, use the following format:

https://notecdn.vercel.app/fetch/{script-path}

Replace {script-path} with the path to the script you want to fetch.

Using Minified Scripts

To fetch minified versions of scripts, use the .min suffix in the script file name. Minified scripts are optimized for performance by reducing the file size.

For example, to fetch a minified version of Axios:

https://notecdn.vercel.app/fetch/npm/axios/dist/axios.min.js

Ensure you use the correct path to the minified version of the script.

Since NoteCDN is based on jsDelivr, it will automatically minify your files using tools like Terser, clean-css, Rollup, or others depending on the type of file you are using.

Examples

Fetching a script from NoteCDN:

https://notecdn.vercel.app/fetch/npm/axios/dist/axios.min.js

This fetches the axios.min.js script from NoteCDN.

Integration in Your Website

To integrate NoteCDN in your website:

  1. Construct the URL with the script path you need to fetch.
  2. Use this URL in your HTML <script> tag's src attribute.
  3. Ensure the path points to the correct script location.
  4. Example:
    5. <script src="https://notecdn.vercel.app/fetch/npm/axios/dist/axios.min.js"></script>

Example Code Block

Here is an example of how to integrate Axios in your HTML file using NoteCDN:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Website</title>
</head>
<body>
    <!-- Axios from NoteCDN -->
    <script src="https://notecdn.vercel.app/fetch/npm/axios/dist/axios.min.js"></script>

    <script>
        // Example usage of Axios
        axios.get('https://api.example.com/data')
            .then(function (response) {
                console.log(response.data);
            })
            .catch(function (error) {
                console.error('Error fetching data:', error);
            });
    </script>
</body>
</html>
