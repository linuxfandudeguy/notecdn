NoteCDN Script Fetching Documentation

Welcome to the NoteCDN script fetching documentation. NoteCDN fetches scripts using a path-based method similar to jsDelivr.

How to Use NoteCDN

To fetch scripts using NoteCDN, use the following format:

https://notecdn.vercel.app/fetch/{script-path}

Replace {script-path} with the path to the script you want to fetch.

Examples

Fetching a script from NoteCDN:

https://notecdn.vercel.app/fetch/npm/axios/dist/axios.min.js

This fetches the axios.min.js script from NoteCDN.

Integration in Your Website

To integrate NoteCDN in your website:

  1. Construct the URL with the script path you need to fetch.
  2. Use this URL in your HTML <script> tag's src attribute.
  3. Ensure the path points to the correct script location.
  4. Example:
    5. <script src="https://notecdn.vercel.app/fetch/npm/axios/dist/axios.min.js"></script>

For more information and examples, visit JSdelivr.
